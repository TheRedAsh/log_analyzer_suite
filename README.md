# 📊 Log Analyzer Suite

> A lightweight, efficient Python suite designed to automatically **generate** realistic mock log data and **parse/analyze** system or server logs for quick insights and security audits.

---

## 🚀 Features

- ⚡ **Automated Log Generation:** Create realistic sample log files for testing and practice.
- 🔍 **Fast Parsing:** Extract critical events, status codes, and error traces efficiently.
- 🛡️ **Cybersecurity Ready:** Ideal for SOC analysts, threat hunters, and system administrators testing log pipelines.

---
